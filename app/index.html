Мобильное приложение для водителей
Мобильное приложение для водителей позволяет держать свой баланс под полным контролем и снять деньги в любое удобное время. Основная задача моментальный вывод денег с баланса профиля водителя, на банковскую карточку водителя.
Смотреть презентацию
Технологии
ASP.net, C #, MVC, Microsoft SQL server, Java
Ключевые модули приложения:
  • Авторизация, Регистрация, Смена пароля
  • Подтверждение СМС, Пин-код для дальнейшего входа
  • Главная страница с данными профиля и балансом, а также возможность выбора карты и указания суммы перевода
  • История переводов

результат

1. Быстрый вывод денег из таксопарка на карту. Укажите свои банковские реквизиты в мобильном приложении и получайте быстрые выплаты прямо на карту. Мы предоставляем возможность вывода денег с яндекс такси

2. При добавление карты есть валидация карты через сервис (валиден ли номер карты). Можно выбирать или удалять карты.
